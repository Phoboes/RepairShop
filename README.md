Njs ticketing app from [Dave Gray](https://www.youtube.com/@DaveGrayTeachesCode).

Stack:
- Next.js 15 && React 19
- Shadcn UI
- Tailwind
- TypeScript
- React Hook Form
- Zod - Schema validation
- Kinde Auth - Login
- Sentry - Error logging
- Tanstack - Table management & utilities
- Neon - Database
- Vercel - Hosting
